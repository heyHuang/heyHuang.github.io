前言

网站构建基础：域名+服务器
本网站域名 www.hqyblog.top 购买于阿里云，为期一年。服务器是借助于Github网站，优点免费，缺点加载有点慢。如果是像我一样只是随意尝试构建一个blog，可以采取这种方式。但是如果需要构建真正的网站，还是需要花一笔钱去阿里云购买.com域名和服务器。

网站页面：Hexo
网上有很多类似的构建页面的工具，但是经过各种对比，对于我来说还是Hexo比较容易上手。这里提醒一下使用Hexo对纯小白有点困难，新手可以上网查询其他容易操作的框架。

npm git node.js 下载并最好确保更新到最新版本

进阶步骤

一、修改域名

（1）首先购买一个域名，我是直接上阿里云选了一个比较便宜的域名，如果是.com这种费用就会高一点。购买成功之后，直接在阿里云里面操作，打开域名控制台，打开刚才购买的域名的解析，添加记录：记录类型选择CNAME，主机记录填写www，记录值填写heyHuang.github.io
（2）然后打开github中heyHuang.github.io仓库，添加文件，命名为CNAME，内容则你申请的域名（www.hqyblog.top ）
等一会儿，打开www.hqyblog.top 网页，此时域名已经修改成功。
